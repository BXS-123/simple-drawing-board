# 画板应用

一个具有用户认证和作品管理功能的网页版简易画板工具，可以进行各种基本绘图操作，并保存和管理个人作品。

## 如何启动

### 方法一：使用批处理脚本（推荐）

1. 找到 `start_app.bat` 文件
2. 双击运行该文件
3. 系统会自动在默认浏览器中打开画板工具登录页面

### 方法二：手动打开

1. 在文件资源管理器中找到 `login.html` 文件
2. 双击该文件或右键选择"在浏览器中打开"

## 主要功能

### 用户认证模块
- **登录页面**：提供用户名/密码输入框、登录按钮、注册链接
- **注册功能**：新用户可创建账号
- **权限控制**：未登录用户仅能访问登录/注册页面

### 作品管理模块
- **我的作品页面**：展示当前用户所有已保存的作品列表
- **作品操作**：对每个作品提供查看、编辑、删除功能按钮
- **新建作品**：提供新建作品按钮，点击进入画布系统

### 绘图功能
- **绘图工具**: 画笔、橡皮擦、矩形、圆形、直线、三角形、文字
- **颜色选择**: 自定义颜色选择器
- **笔触设置**: 可调节线宽、透明度、笔触样式
- **画布操作**: 新建画布、清空画布、缩放画布、全屏显示
- **编辑功能**: 撤销、重做、保存图像
- **辅助功能**: 网格显示、鼠标位置显示

### 画布系统集成
- **保存功能**：在画布系统中完成创作后，点击"保存"可将作品存储到用户的作品列表
- **退出功能**：画布系统中提供"退出编辑"按钮，点击后返回"我的作品"页面

### 个人中心模块
- **入口**：从"我的作品"页面可进入个人中心
- **账号管理**：提供修改密码功能
- **账号注销**：提供账号注销功能（需谨慎操作的确认机制）

## 快捷键

- **工具选择**: B (画笔), E (橡皮擦), R (矩形), C (圆形), L (直线), T (文字)
- **编辑操作**: Ctrl+Z (撤销), Ctrl+Y (重做)
- **文件操作**: 
  - Ctrl+S (保存图片到本地)
  - Ctrl+Shift+S (保存作品到云端)
  - Ctrl+N (新建画布)
  - Ctrl+Shift+X (清空画布)
  - Ctrl+Q (退出编辑)

我们对代码进行了以下修复，确保画板工具能够正常使用：

5. **添加了启动脚本**：创建了批处理脚本，方便用户直接启动应用

## 注意事项

- 建议使用现代浏览器（Chrome、Firefox、Edge等）以获得最佳体验
- 所有绘图操作都保存在浏览器内存中，请及时保存您的作品
- 如需扩展功能或修复更多问题，请查看源代码或联系开发者

## 页面跳转逻辑

- 登录成功 → 进入"我的作品"页面
- "新建作品" → 进入画布系统
- "退出编辑" → 从画布系统返回"我的作品"页面
- 个人中心入口 → 从"我的作品"页面进入
- 密码修改/注销完成 → 返回"我的作品"页面或登录页面（根据操作类型）
